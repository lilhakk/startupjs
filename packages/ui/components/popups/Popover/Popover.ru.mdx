# Popover
Всплывающее окно для показа дополнительного контента, связанного каким-то образом с определённым элементом на странице и появляющееся при нажатии на него.

```jsx
import { Popover } from '@startupjs/ui'
```

## Инициализация

Перед использованием нужно настроить [Portal](/docs/components/Portal)

## Видимость

### Простой пример

```jsx example
const contentStyle = { paddingLeft: 24, paddingRight: 24, paddingTop: 16, paddingBottom: 16 }

function renderContent () {
  return pug`
    View(style=contentStyle)
      Text Контент
  `
}

return pug`
  Popover(
    style={ width: 24 }
    renderContent=renderContent
  )
    Icon(icon=faUserCircle size="xl")
`
```

### С использованием visible

```jsx example
const [visible, setVisible] = useState(false)

const contentStyle = { paddingLeft: 24, paddingRight: 24, paddingTop: 16, paddingBottom: 16 }

function renderContent () {
  return pug`
    View(style=contentStyle)
      Text Контент
  `
}

return pug`
  Popover(
    visible=visible
    style={ width: 24 }
    renderContent=renderContent
    onDismiss=v=> setVisible(false)
  )
    TouchableOpacity(
      onPress=()=> setVisible(true)
    )
      Icon(icon=faUserCircle size="xl")
`
```

### С использованием $visible

```jsx example
const [, $visible] = useValue(false)

const contentStyle = { paddingLeft: 24, paddingRight: 24, paddingTop: 16, paddingBottom: 16 }

function renderContent () {
  return pug`
    View(style=contentStyle)
      Text Контент
  `
}

return pug`
  Popover(
    $visible=$visible
    style={ width: 24 }
    renderContent=renderContent
    onDismiss=()=> $visible.set(false)
  )
    TouchableOpacity(
      onPress=()=> $visible.set(true)
    )
      Icon(icon=faUserCircle size="xl")
`
```
